# Доступен

Статический сайт для приёма заявок через Formspree и публикации ответов на
GitHub Pages.

## Как опубликовать ответ

После отправки формы в Formspree придёт поле `request_id`, например:

```text
5b16b590-9027-4d0f-8adc-f81570fb5c7d
```

Добавьте ответ в `resolutions.json`, используя этот UUID как ключ:

```json
{
  "5b16b590-9027-4d0f-8adc-f81570fb5c7d": {
    "answer": "Заявка рассмотрена. Здесь находится ответ клиенту."
  }
}
```

Допустима и сокращённая запись:

```json
{
  "5b16b590-9027-4d0f-8adc-f81570fb5c7d": "Текст ответа клиенту."
}
```

После коммита и публикации GitHub Pages ответ станет доступен по ссылке:

```text
https://<username>.github.io/<repository>/?request=<uuid>
```

Не публикуйте в ответах персональные, платёжные или другие конфиденциальные
данные: `resolutions.json` доступен любому посетителю сайта.

## Локальный запуск

Запустите любой статический HTTP-сервер из корня проекта, например:

```sh
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## GitHub Pages

В настройках репозитория откройте **Settings → Pages**, выберите
**Deploy from a branch**, ветку `main` и каталог `/ (root)`.
